## 프로젝트 목적
* 울산광역시 청소년활동진흥센터 울산 청소년 페스티벌 학과체험 부스팀 지원
* 울산대학교 IT융합학부 대표 가온누리 "컴퓨터야 놀자"팀 
* 부스에 사용할 OX퀴즈 게임 프로젝트를 목적으로 제작하였습니다.

## 프로젝트 소개
* 간단한 상식형 OX 퀴즈 게임입니다.
* 다양한 청소년들이 오는 만큼 난이도는 낮게 설정하였습니다.
